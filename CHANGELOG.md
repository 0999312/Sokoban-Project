# Changelog

## 1.0.1

- Replaced the project UI font with `MiSans-Semibold` and updated Credits font attribution.
- Updated Credits to rename `mc_game_framework` to `Minecraft-Style-Framework` and credit its author.
- Improved level editor board panning so middle mouse drag and `Space + Left Mouse` pan reliably without interfering with the editor UI.
- Added regression coverage for editor board panning and input binding domain behavior.
- Added GitHub Actions automation for publishing release builds to itch.io via `butler`.
